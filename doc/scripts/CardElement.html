Custom web component representing a card.

Constructor

new CardElement(card, draggableopt)

Initializes the CardElement and sets up its structure and event listeners.
Parameters:
Name Type Attributes Default Description
card Card The backend Card object associated with this element.
draggable boolean <optional>
 true Whether the card is draggable. Defaults to true.
Properties:
Name Type Description
_container HTMLElement The main container for the card.
_cardInner HTMLElement The inner container for the card's 3D flip effect.
_cardFront HTMLElement The front face of the card.
_cardBack HTMLElement The back face of the card.
_tooltip HTMLElement The tooltip element for displaying additional information.
_dragging boolean Indicates if the card is currently being dragged.
_offset object The offset of the card from the mouse cursor during drag.
_wasDragged boolean Indicates if the card was dragged before flipping.
_dragStartX number The X coordinate where the drag started.
_dragStartY number The Y coordinate where the drag started.
_DRAG_THRESHOLD number The pixel threshold to consider a drag action.
_lastClientX number The last X coordinate of the mouse during drag.
_tiltFactor number The factor by which the card tilts based on mouse movement.
_dragPreparationState object Stores the initial state of the card before dragging starts.
_preDragTransform string The original transform style of the card before dragging.
_card string Mirror of the backend Card object associated with this element.
_draggable boolean Indicates if the card is draggable.
